Protractor tester

Test protractor selectors from your browser

Instructions

The extension includes two components:

  • A popup to enter protractor locators and expressions.
  • A developer tools to suggest locators for the selected element.

To test locators you need to start run Protractor's element explorer.

Start the selenium standalone server
$ protractor/bin/webdriver-manager start
Open the element explorer
$ protractor/bin/elementexplorer.js

Using the popup

You can use the popup to enter Protractor selectors or execute commands that you would normally enter in the element explorer.

When you enter a locator starting with by. (e.g. by.css, by.model, by.binding) then it will be executed as a count expression element.all(<your locator>).count() by the element explorer.

Using the developer tools

To use the developer tools you need to open a new browser window. Protractor's element explorer will close the developer tools every time you issue a command.
